Este videojuego consiste en mover al personaje para recolectar cerezas, las cuales se traducen como puntos en el juego, mientras se evitan enemigos que aparecen en el escenario. A medida que se avanza, los niveles se vuelven más difíciles, ya que aumenta la cantidad de enemigos o la dificultad para esquivarlos. El objetivo de este videojuego consiste en recolectar la mayor cantidad de cerezas (puntos) en el menor tiempo posible.
